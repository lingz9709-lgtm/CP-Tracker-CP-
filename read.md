一、写在前面
你有没有过这样的一瞬间：
刷着刷着抖音，突然看到某对CP的一个片段——心跳漏了一拍，然后你疯狂去翻他们的过去，一路嗑到凌晨三点。
CP文化，从来不只是「嗑糖」那么简单。
那是一种属于无数人的共同情感记忆，是CP粉们心照不宣的暗语，是那些说不清道不明却偏偏就是会心跳加速的感觉。
CP Tracker，就是要把这份「嗑CP」的体验，做成抖音信息流里最让人停不下来的卡片。
不是视频，不是图文，而是一个在你刷到的那一刻就已经「懂你」的内容单元。

在抖音，CP内容从来不缺热度。但绝大多数内容都在「消费」CP，而不是「服务」CP粉。

 五、技术实现
1、cp追踪
This content is only supported in a Feishu Docs
- 先输入 CP 名称 和别名，系统自动拆成关键词（比如主名、简称、话题标签）。
- 用这些关键词去多个平台检索内容（你这套里主要是抖音/微博/LOFTER，也支持扩展）。
- 抓到的原始内容先做一轮“CP相关性过滤”，把无关、弱相关、重复内容先剔掉。
- 对保留下来的高赞视频和评论做结构化提取：识别“场景词”“互动类型”“情绪强度”等关键信号。
- 再把这些信号交给 AI 生成模块，按时间和情感推进拼成完整故事线（并输出报告、前端可视化 JSON）。
